# Tam Korea 웹사이트 프로젝트

이 프로젝트는 React + Vite로 제작된 탐코리아(Tam Korea) 공식 웹사이트입니다.

## 📁 주요 배포/설정 가이드
서버 업로드 및 배포와 관련된 자세한 문서는 아래 파일들을 참고해 주세요.

- **[통합 배포 가이드 (추천)](./VERCEL_FULL_GUIDE.md)**: GitHub와 Vercel을 연동하여 자동 배포하고, 가비아 도메인을 연결하는 방법 (가장 추천)
- **[가비아 FTP 배포 가이드](./GABIA_DEPLOY_GUIDE.md)**: 가비아 웹호스팅을 이용하는 경우 FTP로 직접 파일을 올리는 방법
- **[GitHub 개발 프로세스](./GITHUB_WORKFLOW_GUIDE.md)**: 코드를 수정하고 GitHub에 올리는 흐름 설명

## 💻 개발 및 실행 방법

### 1. 설치 (Installation)
```bash
npm install
```

### 2. 개발 서버 실행 (Run Dev)
내 컴퓨터에서 미리보기를 할 때 사용합니다.
```bash
npm run dev
```

### 3. 빌드 (Build)
서버에 올릴 배포용 파일(`dist` 폴더)을 생성합니다.
