# 웹사이트 배포 가이드

전체 프로젝트 빌드가 완료되었습니다. 아래 순서대로 서버에 업로드하시면 됩니다.

## 1. 배포 파일 위치 확인
새로 생성된 배포용 파일들은 **`dist`** 폴더 안에 있습니다.
- **경로**: `d:\01 work\gravity\dist`

## 2. 서버 업로드 방법 (FTP / 파일질라 등)

1. **서버 접속**: 사용하시는 FTP 프로그램으로 웹 서버에 접속합니다.
2. **웹 루트 이동**: 서버에서 홈페이지 파일이 들어가는 기본 폴더(`public_html` 또는 `www`)로 이동합니다.
3. **업로드 (중요)**: 
   - 내 컴퓨터에서 `dist` 폴더 **안으로 들어갑니다.**
   - 폴더 안에 있는 **모든 파일과 폴더**(`index.html`, `assets`, `partners` 등)를 전체 선택합니다.
   - 선택한 항복들을 서버의 웹 루트 폴더로 **드래그하여 업로드(덮어쓰기)** 합니다.

> **주의**: `dist` 폴더 자체를 통째로 올리시면 안 됩니다. 반드시 폴더 **내용물**을 꺼내서 올려주세요.

## 3. 확인
업로드가 완료되면 홈페이지 주소로 접속하여 변경 사항을 확인합니다.
만약 예전 화면이 계속 보인다면, 키보드의 `Ctrl` + `Shift` + `R` 키를 동시에 눌러서 강력 새로고침을 하세요.
