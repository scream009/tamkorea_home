# 가비아(Gabia) 웹호스팅 배포 가이드

도메인(`tamkorea.com`)과 함께 가비아 웹호스팅을 이용하시는 경우, **FTP**를 통해 파일을 업로드해야 합니다.
(만약 호스팅 없이 도메인만 구매하셨다면, Vercel을 사용하는 것이 더 유리합니다. 이 가이드는 **가비아 웹호스팅 상품**을 보유한 경우입니다.)

## 1. 사전 준비 (필수 파일 생성 완료)
React 사이트는 페이지 이동 시 "404 에러"를 방지하기 위해 설정 파일(`.htaccess`)이 필요합니다.
방금 프로젝트의 `public` 폴더와 `dist` 폴더에 이 파일을 생성해 두었습니다.

## 2. 배포 파일 위치
업로드해야 할 최종 파일들은 아래 폴더에 있습니다.
- **경로**: `d:\01 work\gravity\dist`
- **내용물**: `.htaccess`, `index.html`, `assets` 폴더, `partners` 폴더 등.

## 3. 업로드 순서 (파일질라 사용 권장)

1. **FTP 접속**:
   - **호스트**: 가비아에서 안내받은 FTP 주소 (예: `tamkorea.com` 또는 IP주소)
   - **사용자명/비밀번호**: 가비아 호스팅 관리 페이지에서 설정한 접속 정보
   - **포트**: 21 (기본)

2. **폴더 이동**:
   - 서버 쪽(오른쪽 화면)에서 `public_html` 또는 `www` 폴더로 들어갑니다. (루트 디렉토리)
   - 기존에 있는 `index.html`이나 `welcome.html` 등 기본 파일은 **삭제**하거나 백업합니다.

3. **파일 업로드**:
   - 내 컴퓨터(왼쪽 화면)에서 `dist` 폴더 안으로 들어갑니다.
   - **숨김 파일**인 `.htaccess`를 포함하여 **모든 파일과 폴더**를 전체 선택합니다.
   - 오른쪽 서버 화면으로 드래그하여 업로드합니다.

## 4. 접속 확인
- `http://www.tamkorea.com` 으로 접속하여 화면이 잘 나오는지 확인합니다.
- 메뉴를 클릭해서 페이지 이동이 잘 되는지, 새로고침 했을 때 오류가 없는지 확인합니다.

---
**참고**: `.htaccess` 파일이 없으면 새로고침 시 페이지를 찾을 수 없다는 오류가 발생합니다. 반드시 함께 업로드해 주세요!
